# Keyphoria Website

個人のウェブサイト「Keyphoria」のフロントエンドリポジトリです。

## 概要

このプロジェクトは、「鵺帝」氏によって作成された個人ブログサイトです。HTML, CSS, JavaScriptで構築されており、シンプルなデザインといくつかのインタラクティブな機能を備えています。

### ✨ 主な機能

-   **ナビゲーションメニュー**: レスポンシブ対応の開閉式メニューを実装しています。
-   **アクセスカウンター**: サイトへの訪問者数を表示します。
-   **背景色変更機能**: ユーザーがカラーピッカーを使って自由にページの背景色を変更できます。
-   **Google Analytics連携**: サイトのアクセス解析のためにGoogle Analyticsが導入されています。

## 技術スタック

-   **HTML5**
-   **CSS3**
    -   [Ress.css](https://github.com/filipelinhares/ress): CSSリセットライブラリ
    -   [Google Fonts](https://fonts.google.com/): カスタムフォント
-   **JavaScript**

---

## セットアップ

1.  このリポジトリをクローンまたはダウンロードします。
2.  `index.html` ファイルをブラウザで開くと、ウェブサイトが表示されます。

特別なビルドツールや依存関係のインストールは不要です。
